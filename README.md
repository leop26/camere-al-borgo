# Camere al Borgo – Mobile-First Website

## Overview

**Camere al Borgo** is a mobile-first website designed to present a charming guesthouse in Italy.  
The site is fully optimized for smartphones, offering visitors easy access to:

- Room details  
- Pricing  
- Local attractions  
- Booking information  
- Legal notices  

> ⚠️ **Note:** The desktop version is still in development. Desktop users currently see a message recommending smartphone access for the best experience.

---

## Features

- Mobile-first responsive design (**desktop version in progress**)  
- Multi-language support: Italian, German, English  
- Interactive burger menu for intuitive navigation  
- Room details, pricing, local attractions, and FAQs  
- Booking links to external platforms  
- Contact section with email, phone, and Google Maps integration  
- Legal and privacy pages  

---

## Pages

- `index.html` – Home  
- `rooms.html` – Room overview  
- `vacation.html` – Local attractions  
- `prices.html` – Pricing information  
- `FAQ.html` – Frequently asked questions  
- `booking.html` – Booking page  
- `impressum.html` – Legal notices  
- `privacy-policy.html` – Privacy information
